Detailed Description

Main header file for Exasol error-reporting-cpp.

Include this file in your project to use the error-reporting-cpp macros.

Note
Declare MAIN_ERROR_REPORTING_CPP before including this file in your project in exactly one cpp file. This is where the global variables will be stored.
Declare ERROR_MESSAGE_COLLECTION before including this file in your project in any cpp file, if you want to support collection of all error codes, see error_reporting::error_message_declaration_container::collect_error_messages_as_json()
